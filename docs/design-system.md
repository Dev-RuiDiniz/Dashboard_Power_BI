# Design System

## Estado atual

Existe uma base visual funcional no frontend com componentes de UI reutilizáveis e uma página dedicada em `/design-system`.

## Objetivo atual

O design system serve como fundação da interface, não como biblioteca completa publicada separadamente.

## Evidências

- `apps/web/src/app/design-system/page.tsx`
- `apps/web/src/components/ui`

## Limitações

- não há pacote compartilhado ativo em `packages/ui`;
- tokens e padrões visuais ainda vivem principalmente dentro do app web;
- o escopo V1 descreve uma experiência mais ampla do que a base realmente disponível hoje.
